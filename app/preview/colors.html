How to build your theme faster ?

We first developped this bootsrap theme editor for our own purpose to create theme a lot faster. Here are some technics to become a bootstrap theme chef with this tool.

1. Awesome typeahead

Find as you type all defined variables started with an @ and all the Less function as darken(@color,10%).

2. Reusable variables

Configure the mains colors and the new variables that we created "Brand Colors" and reuse it in order to have a smooth theme.

3. Contextual preview

Change variables, apply and you will get the result on the content on the right. No scroll to check what you have changed.

4. Colorpicker

Colorpicker to help you to find lovely colors.
